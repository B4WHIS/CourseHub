# Backend API

This folder is reserved for the backend API implementation.

For this project, we are using mock data in the frontend (`/frontend/src/data/`).

When you're ready to implement a real backend, consider:

- Express.js, Fastify, or NestJS for Node.js
- Python with FastAPI or Django for Python
- Go with Gin or Fiber for Go

Move your backend files here when ready.
